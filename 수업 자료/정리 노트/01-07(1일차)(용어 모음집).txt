데이터베이스(DB)

개념 

- 넓은 의미로는 컴퓨터 안에 기록된 "모든 정보"를 말함

- 일반적인 의미로는 특정 데이터를 확인하고 싶을 때, 간단하게 찾을 수 있는 형태(파일 관리자처럼)(비슷하자너)(일종의 데이터베이스 접근이자너)

- ★ 영구적 보관 ★ (강제로 delete(삭제) 하지만 않으면 영구적으로 보관됨)

- 일반적으로 (DB // DATA BASE)라고 씀

- 데이터베이스는 디스크(ssd 등 모든 디스크)에 정리되어 저장된 데이터의 집합(모여있는것)

- 이런 데이터베이스를 효율적으로 관리(입력, 수정, 관리, 삭제, 변경 등등)하는 프로그램(DataBase Management System// DBMS)을 데이터베이스 관리 시스템 이라고 함

- DBMS(데이터베이스 관리 시스템)을 사용하면 다음과 같은 장점이 있음
	- 생산성 : 반복적인 데이터 검색, 추가, 삭제등의 작업을 효율적이게 사용가능
	- 가능성 : 대용량 데이터베이스의 고속 검색등 다양한 기능 사용 가능
	- 신뢰성 : 백업 및 복구등 기능을 제공하여 신뢰성을 줌

- RDBMS(Relationmal DataBase Management System) 제품 종류
	- Oracle : 가장비쌈(기능 많음) 오라클사에서 개발
	- DB2 : IBM에서 개발, 거의 사용 X
	- SQL Server : 마이크로소프트에서 개발, 윈도우에서만 사용 가능
	- PostgreSQL : 오픈소스(무료)
	- MySQL : 오픈소스(무료)였으나, 현재는 오라클에서 관리, 많이 사용 ///// 이거 쓸꺼임
	- MariaDB : MySQL의 쌍둥이로 오픈소스(무료), 많이 사용
	- SQLite : 핸드폰에 다 들어있음 // 소형기기 전용 DBMS

- 데이터베이스 관리 시스템과 데이터를 검색, 추가, 삭제 등 작업에 수행함에 있서 ★ SQL(DBMS 사용할때 쓸 프로그래밍 언어) ★이라는 언어 사용

- 특히, 관계(relation(테이블))형 데이터(RDBMS)에서 많이 사용

- SQL(Structure(구조) Qurey Language(언어) : DBMS 프로그래밍 언어 (PPT에는 4개로 나눠져있지만 그것을 모두 SQL언어에 들어가는 것이다 // 한마디로 SQL을 4종류로 나눈것)(수업자료 PPT 7쪽 참고)

- DBMS종류 : 수업자료 PPT 8쪽 참고 

- 테이블 : Excel(엑셀)이나 한셀을 사용할때 보이는 sheet와 같다. (그나까 말이야 윈도우로 보면 한쪽 모니터에는 유튜브를 실행했눈데 다른 한쪽에는 소설을 뛰어둔거 비슷함) (대충 알아듯것지)

SQLD 자격증

1. 국가자격증 : 기사(정보처리기사 등)  (나라에서 직접 관리하는 자격증)
2. 국가공인자격증 : SQLD (국가가 인증해주는 자격증 국가자격증과 비슷한 것임 but 민간기업이 관리함)
3. 민간자격증 : 선물포장 (민간기업이 인증해주는 자격증 // 국가가 인증해주지는 않음)

SQL

- 열/컬럼/필드의 데이터형이 존재

- 숫자형, 문자형, 날짜형, 그리고 값이 없는 NULL값이 존재

- 문자형에는 고정형, 가변형이 있음
	- char형 : 고정된 길이를 가지는 문자저장
	- varchar형 : 최대 길이는 고정이나, 작은 길이의 문자가 들어오면 저장 공간 크기가 줄어듬
